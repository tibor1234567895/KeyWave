# Privacy Policy

KeyWave is built with privacy as a core principle.

## What KeyWave Collects
- Nothing. The app does not collect, store, or transmit personal data.

## Permissions
KeyWave only requests permissions required for its core features:
- Accessibility Service: Detects volume button gestures.
- Notification Listener: Identifies the active media session.
- Vibrate: Provides haptic feedback on successful actions.

## Network Access
KeyWave does not request the Internet permission and cannot send data anywhere.

## Data Storage
All settings are stored locally on your device.

## Contact
If you have questions, open an issue in the project repository.
